COMANDOS RECOMENDADOS

Si copias solo los archivos corregidos sobre tu proyecto actual:
1. php artisan optimize:clear
2. php artisan migrate
3. php artisan db:seed
4. php artisan serve

Si usas la carpeta completa ligera sin vendor:
1. composer install
2. php artisan key:generate
3. npm install
4. npm run build
5. php artisan migrate --seed
6. php artisan serve

CAMBIOS PRINCIPALES

resources/views
- Mantengo el diseño original.
- He añadido comentarios Blade con frases sencillas para explicar cada bloque.
- La calculadora IMC ahora valida datos realistas:
  peso entre 20 y 300 kg
  altura entre 100 y 250 cm
  así no acepta casos absurdos como 890 cm.
- El formulario de contacto ya no apunta a "#".
  Ahora manda los datos a Laravel, valida los campos y muestra un mensaje de éxito.
- Los enlaces que estaban sin destino ahora usan rutas reales.
- El panel admin conserva el estilo, pero ahora los botones Editar y Borrar funcionan.

routes/web.php
- He ordenado las rutas por bloques:
  rutas públicas
  login/logout
  zona admin
- He añadido nombres a las rutas para no escribir URLs sueltas por todo el proyecto.
- He agrupado las rutas admin con middleware auth.

app/Http/Controllers
- AdminController:
  crear, editar, actualizar y borrar suplementos.
  La validación está separada en una función para no repetir código.
- AuthController:
  valida email y contraseña antes de intentar iniciar sesión.
  mantiene el email escrito si hay error.
- ImcController:
  solo carga la vista de la calculadora, el cálculo está en JavaScript.

database/seeders
- DatabaseSeeder ahora llama a UserSeeder y SuplementoSeeder.
- UserSeeder crea el usuario admin.
- SuplementoSeeder crea suplementos de ejemplo sin duplicarlos.
